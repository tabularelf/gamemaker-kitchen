Another GUI library of mine...
Unlike [GMUI](<https://github.com/erkan612/GMUI>) this one is Retained-Mode, but what makes it special is that its canvas/surface based hierarchial.
It gives you 100% control over the GUI not just styling but also behaviours.
It has a deep learning curve, its a bit tricky to learn. I made it before [GMUI](<https://github.com/erkan612/GMUI>) and currently im more focused on it(but its still ready to be used in a game project)

Here's the github repo https://github.com/erkan612/MajorGUI_GML

Feel free to ask if you encounter any issues or have struggles with it!
