# [FauSet](https://github.com/tinkerer-red/FauSet)
FauSet is a lightweight garbage collection wrapper for GameMaker’s built-in data structures. It helps prevent memory leaks from forgotten ds_* objects by automatically cleaning up references when they expire or lose ownership.
